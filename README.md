## Development

### Clone the repo

```git clone https://github.com/csaf-poc/csaf_webview.git```

### `cd` into app directory

```cd csaf_webview```

### Install dependencies

```npm install```

### Run development server and `--open` a browser.

```npm run dev -- --open```
